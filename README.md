# React + Node & Express 🚀

Just a really simple React + Node & Express application with a monorepo format

## How to run? 🙋‍♂️

1. Download or clone this repo.

2. Install dependencies.

```js
npm install
```

3. Run the server.

```js
 npm run dev
```

4. Run the Frontend

```js
 npm run start
```
